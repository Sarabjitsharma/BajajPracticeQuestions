# Coding Problems Collection

This repository contains a collection of coding problems solved in C++, organized by days and categories. These problems are primarily from LeetCode, Codeforces, and other coding platforms, focusing on algorithms, data structures, and problem-solving techniques.

## Structure

The problems are organized into the following folders:

### Day-1: Sliding Window Techniques
- **FindPairWithKSum.cpp**: Find a pair of elements in an array that sum to a given value K.
- **FixedSizeSlidingWindow.cpp**: Implement sliding window problems with a fixed window size.
- **MaxSubArray.cpp**: Find the maximum subarray sum (Kadane's algorithm variant).
- **SlidingWindowTwoPointers.cpp**: Sliding window problems using two pointers.
- **VariableSizeSlidingWindow.cpp**: Sliding window with variable window sizes.
- **VariableSizeSubarray.cpp**: Find subarrays with variable sizes based on conditions.

### Day-2: Array Problems
- **EquilibiriumIndex.cpp**: Find the equilibrium index in an array where sum of left equals sum of right.
- **RangeSum.cpp**: Compute range sum queries on arrays.
- **SplitArray.cpp**: Split an array into subarrays with specific properties.

### Day-3: Search and OOP Concepts
- **BinarySearch.cpp**: Implement binary search algorithm.
- **Inherit.cpp**: Demonstrate inheritance in C++.
- **MethodOver.cpp**: Method overloading examples.
- **OverrideMethod.cpp**: Method overriding examples.

### Day-4: Advanced Array Algorithms
- **Circular.cpp**: Problems involving circular arrays.
- **Kadane.cpp**: Maximum subarray sum using Kadane's algorithm.

### Day-5: String Manipulation (LeetCode Problems)
- **PermutationInString567.cpp**: LeetCode 567 - Permutation in String.
- **RansomNote383.cpp**: LeetCode 383 - Ransom Note.
- **ReverseString344.cpp**: LeetCode 344 - Reverse String.
- **ValidAnagram242.cpp**: LeetCode 242 - Valid Anagram.
- **ValidPalindrome125.cpp**: LeetCode 125 - Valid Palindrome.
- **ValidPalindromeII680.cpp**: LeetCode 680 - Valid Palindrome II.

### DAY-6: Linked Lists and Strings
- **AddTwoNumbers.cpp**: LeetCode 2 - Add Two Numbers (linked list addition).
- **lps.cpp**: Longest Palindromic Substring or similar string problems.

### DAY-7: Miscellaneous
- **firstUniqChar.cpp**: LeetCode 387 - First Unique Character in a String.
- **numOfSubArray.cpp**: Count number of subarrays with certain properties.

## How to Run

Each C++ file contains a main function for testing. Compile and run using:

```bash
g++ FolderName/FileName.cpp -o FileName
./FileName
```

For example:

```bash
g++ Day-1/MaxSubArray.cpp -o MaxSubArray
./MaxSubArray
```
